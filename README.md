# 🤖 AskMeGPT

AskMeGPT is an AI chatbot inspired by ChatGPT, built using the MERN stack and Groq API. It supports chat history, Markdown rendering, code syntax highlighting, and chat deletion.

## 🌐 Live Demo

- Frontend: https://ask-me-gpt-one.vercel.app
- Backend: https://askmegpt-backend.onrender.com

## ✨ Features

- 🤖 AI-powered chatbot
- 💬 Create and manage conversations
- 🗑️ Delete chats
- 📝 Markdown support
- 💻 Code syntax highlighting
- 📚 Persistent chat history
- 📱 Responsive UI

## 🛠️ Tech Stack

- React.js
- Vite
- Node.js
- Express.js
- MongoDB Atlas
- Groq API

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/KammarapalliMeghana/AsKMeGPT.git
cd AskMeGPT
```

### Backend

```bash
cd Backend
npm install
npm start
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

## 🔑 Environment Variables

### Backend (`Backend/.env`)

```env
MONGODB_URI=your_mongodb_uri
GROQ_API_KEY=your_groq_api_key
```

### Frontend (`Frontend/.env`)

```env
VITE_API_URL=http://localhost:8080
```

For production:

```env
VITE_API_URL=https://askmegpt-backend.onrender.com
```
